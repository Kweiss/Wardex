---
name: wardex-status
description: Show the current Wardex security status
---

Check the Wardex security system status by calling the `wardex_get_status` tool.

Display the results in a clear format:
- Enforcement mode (guardian/copilot/adaptive)
- Whether the system is frozen
- Number of evaluations performed
- Number of transactions blocked
- Number of advisories issued
- Current daily volume
- Signer health status
